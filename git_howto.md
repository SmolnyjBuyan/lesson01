# Подсказка по GIT

Создание репозитория:
```sh
git init
```
Доваление файлу версионности в локальном репозитории:
```sh
git add
```
Фиксирование изменений и их описание:
```sh
git commit -m "Message"
```
Вывод разницы между текущей и зафиксированной версией файла:
```sh
git diff
```
