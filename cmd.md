# Подсказки по командной строке

Команда смены директории
```sh
cd c:\folder_name
```

Команда отражения текущей директории для MacOS, Linux!
```sh
pwd
```

Листинг текущей директории
Windows:
```sh
dir
```

Листинг текущей директории
MacOS, Linux:
```sh
ls
```

Удаление файла в Windows:
```sh
del <filename>
```

Удаление файла в MacOS, Linux:
```sh
rm <filename>
```
